Detailed Description

Defines the main window of the application.

The contents of the main window are :

  • Couple of radio buttons for server start and stop.
  • A list displaying the clients currently connected.
  • A disconnect button for disconnecting the selected client.
  • A button form displaying a 3D demo.

It also implements ActionListener for these elements and WindowListener for graceful termination of the application upon closing of this MainWindow. The window is initialized and rendered at the time of instantiation by the constructor MainWindow(NetworkManager).
