Detailed Description

Utility for the end-to-end security module.

This class provides the following features:

  • 256-bit Elliptic Curve private and public key generation.
  • AES 256-bit plaintext encryption.
  • AES 256-bit cipher text decryption.
  • Self-signed certificate generation.

The private-public key pair and the self-signed certificate is generated by the constructor during instantiation and is stored in a keystore (java.security.KeyStore) maintained by the KeyStoreManager. If a KeyStore already exists, the key pair and the certificate gets loaded directly. The encryption and decryption mechanism is provided by the encryptString(String) and decryptString(String) methods respectively.
