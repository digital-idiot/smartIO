Detailed Description

Implementation of the java.lang.Runnable for the thread responsible for making UDP broadcasts.

This thread continuously sends UDP broadcasts through all available network interfaces of the PC except point-to-point and loopback interfaces. The content of the broadcast data is a JSon-string of a ServerInfo object.

It is started during server start by the method NetworkManager#startServer(). It is stopped during server stop by the method NetworkManager#stopServer(). The interval between two successive broadcast is 1000 ms.

This UDP broadcast is used for device discovery from the Android devices.
