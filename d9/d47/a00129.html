Detailed Description

Manages the different operations on the network.

This class creates a NetworkState which stores the information regarding the network. It provides methods for start and stop of the server. It also provides method for disconnecting a client. These methods modify the NetworkState accordingly.

This class provides back-end of the network operations initiated by the GUI of SmartIO.

◆ stopServer()

void project.pc.net.NetworkManager.stopServer ( ) throws IOException, InterruptedException

Method to stop server.

This is achieved by executing three primary operations :

  • Stopping BroadcastThread by calling BroadcastThread#stopBroadcast().
  • Calling Server#setStopFlag().
  • Disconnecting all clients. This is done by calling disconnect(InetAddress) for every connection stored in the network state.
