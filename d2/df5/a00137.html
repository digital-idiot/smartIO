Detailed Description

Implementation of the java.lang.Runnable for the thread that executes the network operations.

This keeps away the network operations from the Swing Event Dispatch Thread and prevents the GUI from becoming unresponsive. This thread gets started by project.pc.gui.MainWindow while performing the action of starting the server - the first possible network operation.
