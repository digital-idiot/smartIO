Detailed Description

Utility for performing mouse input operations.

It uses a java.awt.Robot object to generate native system input events. This differs from posting events to the AWT event queue or AWT components in that the events are generated in the platform's native input queue.

The move(Quaternion, boolean, float) and move(int, int, float) methods are invoked for moving the mouse pointer on the basis of 3-D and 2-D inputs from the mobile device respectively. The move(Quaternion, boolean, float) method, in turn, calls the MouseMove#moveRelatively(Robot, int, int, float) method. The mouse button inputs are performed by invoking the doOperation(String) method.
