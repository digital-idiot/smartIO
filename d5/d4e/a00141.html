Detailed Description

Server data structure.

It encapsulates the java.net.ServerSocket used for establishing TCP connection with the client.

When the Server starts, the listen() method is called which listens on the port in order to accept incoming client connections. Upon accepting connection it instantiates and starts a ServerThread for each connection.

It also instantiates the project.pc.device.MouseController and project.pc.device.KeyboardController which is used for mouse and keyboard control respectively.
