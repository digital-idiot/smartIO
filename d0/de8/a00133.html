Detailed Description

Stores the state information about the network.

It stores the information about the client connections with the help of two data structures :

  1. A Connection map which is a HashMap from a client Socket to the ServerThread handling that client.
  2. An Address map which is a HashMap from the InetAddress of a client to the Socket.

The methods to manipulate these data structures are provided in the class. The add(ClientInfo, ServerThread) and remove(ClientInfo) methods are invoked by ServerThread once it successfully establishes and closes a client connection respectively.

It also stores a javax.swing.DefaultListModel which is used by project.pc.gui.MainWindow to populate a list displaying the client connections in the GUI.
